# ONNX to TFLite Converter 
Convert your openwakeword generated .onnx File to .tflite File

If you are using the colab from openwakeword and get an error for missing .tflite File
then use this colab to convert your .onnx File to .tflite File!
Original you find here the https://colab.research.google.com/drive/1q1oe2zOyZp7UsB3jJiQ1IFn8z5YfjwEb

Steps:
- download the .onnx File generated by openwakeword
- open file explorer "folder icon" in left sidebar
- click upload button to import your generated .onnx File
- IMPORTANT: in first lines of the script change the Filename to your .onnx File!

Run the Script:
- press the play-button on top to do all tasks
- new generated .tflite File should be vissible in project folder now!
